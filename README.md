# 🚀 Automated Job Data Scraper and Analyzer

A Python-based automation tool that scrapes job listings from websites using Selenium and analyzes the data to generate meaningful insights in structured formats like CSV and Excel.

---

## 📖 Project Overview

This project automates the process of collecting job-related data from online platforms. It extracts key information such as job titles, company names, locations, and required skills, and then processes the data for analysis.

---

## 🛠️ Tech Stack

* Python 🐍
* Selenium 🌐
* Pandas 📊

---

## ✨ Features

* Automated job data scraping
* Handles dynamic websites (JavaScript-based)
* Extracts job title, company, location, and skills
* Data cleaning and preprocessing
* Export data to CSV and Excel
* Easy to extend for analysis

## ⚙️ Installation

### 1. Clone the repository

git clone hhttp://github.com/pamu15/AUTOMATED-JOB-DATA-SCRAPER-AND-ANALYZER/edit/main
cd automated-job-data-scraper-analyzer

### 2. Create virtual environment

python -m venv venv

### 3. Activate environment (Windows)

venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt
